PostPilot

Write once. Post everywhere.

PostPilot is a lightweight, self-hosted social-media autopilot. Compose a post (with AI help if you like), generate an image or short video, then publish or schedule it across Bluesky, Mastodon, X, LinkedIn, Instagram and TikTok — from one place. This page explains exactly how it works, what's reliable, what isn't, and how to test it yourself.

5 platforms automated No build step · single Python process Your accounts stay on your machine

How it works

PostPilot reaches each platform the most reliable way available. There are three lanes:

1 · Official APIs

Bluesky & Mastodon post directly through their public APIs — server-side, instant, rock-solid. No browser needed.

2 · Your real Chrome (CDP)

X, LinkedIn & Instagram have no usable posting API, so PostPilot drives a real Chrome you've logged into, over the DevTools protocol — like a human, but automated.

3 · Manual handoff

TikTok actively blocks automated uploads. PostPilot prepares the video + caption; you do the final tap. Honest about the one wall we can't climb.

Also works

  • • AI captions (one idea → polished, per-platform length)
  • • AI images + slideshow video (vertical MP4)
  • • Media preview before posting
  • • Scheduling that survives restarts
  • • Content Series + calendar (recurring concepts, AI-drafted)
  • • Per-platform results — one failure never blocks the others

Known limits

  • • TikTok: automated upload is blocked by its bot-detection
  • • X / LinkedIn / Instagram need the PostPilot Chrome open + logged in
  • • AI text-to-video is off unless a provider key is set
  • • Web UIs change — a driver may need a selector tweak now and then

How to test everything

  1. 1 · Connect the API platforms

    In Accounts, paste a Bluesky app password and a Mastodon access token. Compose a short post, tick Bluesky + Mastodon, and hit Post now — both should go live instantly. This proves the core pipeline.

  2. 2 · Start the PostPilot Chrome

    For X, LinkedIn & Instagram, open the dedicated browser and log into each site once (logins persist):

    cd postpilot && ./postpilot-chrome

    Keep this window open while posting. It uses a separate Chrome profile, so it won't touch your everyday browser.

  3. 3 · Post to the browser platforms

    Compose a post with an image or video, tick X / LinkedIn / Instagram, and Post now. PostPilot drives your Chrome to publish each one. Watch History for per-platform success/failure (failures save a debug screenshot).

  4. 4 · TikTok (manual)

    PostPilot generates the video + caption; download them from the post and upload on TikTok yourself. Automated upload is blocked on TikTok's side.

  5. 5 · Try the AI + scheduling

    Use Generate caption from a rough idea, generate an image or slideshow, schedule a post for a few minutes out and confirm it fires on its own. Set up a Series to auto-draft recurring posts.

Your data stays with you. PostPilot is self-hosted — it runs as one small process on your own machine. Account credentials are encrypted locally and your browser logins live in the PostPilot Chrome profile on your computer, never on a third-party server.
